/*******************************************************************************
Hardware Connection:
 * PA0: 定位孔传感器-DO
 * PA1: 载带芯片检测传感器-DO
 * PA3: 自锁开关
 * PC13: 系统启动指示灯-LED(+)
 * USART1: 串口通信：PB6(TX)和PB7(RX)（重映射）
 * OLED: 用于显示统计信息: SCL:PB8,SDA:PB9（重映射）
 * PA8: 蜂鸣器-DO
 * PB5: DHT11温湿度传感器-DI
*******************************************************************************/
KEY:
 *  PB0:UP_KEY
 *  PB1:DOWN_KEY
 *  PB10:BACK_KEY
 *  PA2:OK_KEY
/******************************************************************************/
Software Init:
  * PA4:ADC1_IN4(模拟输入)
  * PB6&PB7:UART1-（Remaping）
